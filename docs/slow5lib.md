# slow5lib

## NAME

slow5lib - slow5lib Overview

## DESCRIPTION

slow5lib is a library for reading and writing SLOW5 files.

slow5lib interface consists of following functions:

* slow5_open
    Open a SLOW5 file.
* slow5_close
    Close a  SLOW5 file
* slow5_get
    Fetch a record corresponding to a given read ID.
